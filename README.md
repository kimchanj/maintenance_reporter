# Maintenance Reporter

엑셀 유지보수 내역으로 VPN 작업내역서를 생성하고, 수정된 VPN 작업내역서로 일반 유지보수 내역서를 다시 생성하는 Windows GUI 프로그램입니다.

## 주요 기능
- `진행상태=완료` 데이터만 엑셀에서 반영
- `처리일자` 기준 오름차순 정렬
- VPN 작업내역서(`.docx`) 자동 생성
- 수정된 VPN 작업내역서의 `작업내역` 셀을 읽어 일반 유지보수 내역서(`.docx`) 자동 생성
- 일반 유지보수 내역 변환 시 각 작업 블록의 첫 줄에서 날짜 추출
- 일반 유지보수 내역 변환 시 각 작업 블록의 둘째 줄을 내용으로 그대로 매핑
- 생성 직후 Word 교차검증 수행

## 요구사항
- Python 3.10+
- Windows
- Word 템플릿: `.docx`

## 설치
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## 실행
```bash
python -m app
```

## 작업 순서
1. `유지보수 엑셀`과 `VPN 템플릿`을 선택합니다.
2. `VPN 내역서 생성`을 눌러 VPN 작업내역서를 생성합니다.
3. 생성된 VPN 작업내역서를 Word에서 수정합니다.
4. `수정된 VPN 내역서`와 `일반 템플릿`을 선택합니다.
5. `일반 내역서 생성`을 눌러 일반 유지보수 내역서를 생성합니다.

## VPN 수정본 파싱 규칙
- `작업내역` 셀의 각 작업 항목은 빈 줄로 구분합니다.
- 각 작업 항목은 최소 2줄이어야 합니다.
- 첫 줄: 날짜가 포함된 메타 줄
- 둘째 줄: 일반 유지보수 내역서의 `내용`
- 셋째 줄 이상이 있어도 일반 유지보수 내역서는 둘째 줄만 사용합니다.
- 일반 유지보수 내역서의 `- 처리 완료` 문구는 고정으로 자동 작성됩니다.

## 검증
- 1단계: 엑셀 데이터와 생성된 VPN 작업내역서 비교
- 2단계: VPN 수정본에서 파싱한 데이터와 생성된 일반 유지보수 내역서 비교
