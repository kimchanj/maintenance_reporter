# Maintenance Reporter

엑셀 유지보수 내역을 Word 템플릿에 자동 매핑해 보고서를 생성하는 Windows GUI 프로그램입니다.

## 주요 기능
- Word 템플릿의 `{{placeholder}}` 항목 자동 추출
- 엑셀 헤더와 일치하는 항목만 매핑
- `진행상태=완료` 데이터만 반영
- `처리일자` 오름차순 정렬
- `처리내용`은 첫 문장만 추출해 매핑하며, 첫 문장 내 `[]` 태그/내용은 제거
- 템플릿 2종 자동 처리
  - 행 확장형: 데이터 건수만큼 표 행 추가
  - 단일 셀 누적형(VPN 형식): 한 셀에 여러 건 누적 작성
- 작성 후 엑셀/워드 교차검증(건수 + 내용)

## 요구사항
- Python 3.10+
- Windows
- Word 템플릿: `.docx`

## 설치
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## 실행
```bash
python -m app
```

## 사용 방법
1. 엑셀 파일(`.xlsx`) 선택
2. 워드 템플릿 파일(`.docx`만 지원) 선택
3. `작성` 버튼 클릭
4. 템플릿 폴더에 `원본명_작성완료.docx` 생성

## 입력/처리 규칙
- 필수 필터: `진행상태=완료`
- `진행상태` 컬럼이 없으면 오류
- 템플릿 placeholder 기준으로 엑셀 컬럼 매핑
- `처리내용` 컬럼은 첫 문장만 사용하며, 첫 문장 내 `[]` 태그/내용은 제거
- 매핑되지 않은 placeholder는 빈 값으로 처리

## 검증
작성 직후 자동 검증을 수행합니다.
- 행 확장형: 행/열 단위 값 비교
- 단일 셀 누적형: 누적 셀 텍스트 + 1회성 placeholder 셀 비교
