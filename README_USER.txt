유지보수 보고서 작성기 (배포본)

1) 실행
- MaintenanceReporter.exe 더블클릭

2) 사용 방법
- [Excel] 버튼으로 유지보수 내역(.xlsx)을 선택
- [Word] 버튼으로 유지보수 보고서 템플릿(.docx)을 선택
  ※ 템플릿은 .docx만 지원합니다. (.doc은 Word에서 다른 이름으로 저장 → .docx로 변환 후 사용)
- [작성] 버튼 클릭
- 결과 파일은 템플릿과 같은 폴더에 아래 이름으로 저장됩니다.
  템플릿파일명_작성완료.docx

3) 자주 발생하는 문제
- 템플릿이 .doc인 경우:
  Word에서 열고 '다른 이름으로 저장'으로 .docx로 저장 후 다시 선택하세요.

4) 문의/수정 포인트
- 엑셀 헤더명이 바뀌면 프로그램이 컬럼을 못 찾을 수 있습니다.
  기본 요구 컬럼: 처리일자, 처리내역, 진행상태
